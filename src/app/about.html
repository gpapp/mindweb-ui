MindWeb application

Mindweb is a document outliner application that enables you to create/view/share document outlines* in the form of mindmaps.

There are three ways of displaying the stored data. In the classical tree structure (Outline view), as a mindmap (Mindmap view) and as a list of extracted tasks (Task list view)

It's originated from the Java based Freeplane mindmap tool, do it's perfectly possible to import/export* mindmaps from the Freeplane tool.

It's also possible to connect the Freeplane directly to the service using the provided add-on*

* Most of these features are only planned, and are not yet implemented. Please contact the author for more information.
